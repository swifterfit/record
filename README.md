# record

Daily record

## 每日打卡

- 每日记录拆分为 100 份 Markdown 文件，命名为 `YYYY_MM_DD.md`（起始 2025-12-19，共 100 天）。
- 每份文件包含三项：技术 / 健身 / 英语，仅含标题与“> 细节：”引用行，可直接填充。

### 辅助脚本

- 使用 `python3 daily_entry.py` 运行。
- 流程：输入日期（回车默认今天）→ 依次填写技术/健身/英语细节 → 可选自动 `git add/commit/push`。
- 若对应日期的 MD 不存在会自动按模板创建；已存在则用已有细节作默认值，便于修改。
